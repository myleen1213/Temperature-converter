# TempConvert

## The Basics 

This application is a simple temperature converter that converts a celcius degree to fahreheit or a fahrenheit to a celcius.

## Technology Used

* Language : Kotlin 
* Build : Gradle 
* Android studio


## How to use

* Git clone from command line
* Download the zip file
